YummyDroid - нативное приложение для Android, Android TV, ТВ-приставок, телефонов и планшетов для просмотра аниме с YummyAnime. Приложение рассчитано на универсальное управление: сенсорный экран, мышь, клавиатура, физический пульт, виртуальный пульт, мультимедийные клавиши, PiP и полноэкранный режим.

## Возможности

- Адаптивный интерфейс каталога для телефонов, планшетов и ТВ.
- Поиск, голосовой поиск, сортировка, мультивыбор фильтров, фильтр по меткам пользователя и фильтр доступного офлайн-контента.
- Карточки аниме с рейтингом, просмотрами, постерами, описанием, скриншотами, метаданными, студиями, режиссёрами, жанрами, порядком просмотра, комментариями, оценками, трейлерами, коллекциями и рекомендациями.
- Нативный плеер Media3/ExoPlayer с автоматическим выбором источника, выбором качества, выбором озвучки, пропуском опенинга/эндинга, автопереходом к следующей серии, настройками декодера, PiP-управлением и синхронизацией прогресса просмотра.
- Авторизация, профиль, метки, избранное, оценки, подписки на серии и ссылки на активный рабочий домен сайта.
- Автоматическое переключение доменов сайта и ручное управление списком доменов.
- Офлайн-режим: скачанные серии привязаны к карточке аниме, данные карточки кэшируются, а локальный файл воспроизводится вместо онлайн-потока.
- Фоновые загрузки с системным уведомлением, отдельным разделом загрузок, скачиванием всех серий сразу и выбором качества перед сохранением видео.
- Обновления через GitHub Releases.

## Офлайн-просмотр

При скачивании серии YummyDroid сохраняет карточку аниме, список серий, метаданные и ссылку на локальный медиафайл вместе. Если сайт недоступен, карточку всё равно можно открыть из локального кэша, а скачанная серия запускается из файла на устройстве.

В фильтрах есть пункт `Доступно офлайн`. Если сайт или интернет недоступны, а в приложении есть сохранённые аниме, каталог автоматически переключается на офлайн-список и показывает индикатор `Оффлайн`. Если пользователь вручную включает офлайн-фильтр, индикатор не отображается.

## Интерфейс и управление

Интерфейс сделан для быстрого просмотра каталога и удобной навигации с пульта:

- Разделы главного экрана: каталог, топ, расписание, коллекции, библиотека и загрузки.
- Поиск и фильтры открываются отдельными кнопками и не занимают место постоянно.
- Кнопки серий показывают превью видео и статус доступности офлайн.
- Плеер поддерживает классические иконки переключения серий, фокусируемый таймлайн, ускоренную перемотку, PiP-кнопки и скрытие системных панелей в полноэкранном режиме.

## Установка

Последний APK доступен на странице релизов:

https://github.com/saltek1995/YummyDroid/releases/latest

Файлы APK называются так:

```text
YummyDroid-<версия>.apk
```

## Сборка

Требования:

- Android Studio или установленный Android SDK.
- JDK 21.

Собрать release APK:

```powershell
.\gradlew.bat :app:assembleRelease
```

Проверить компиляцию Kotlin:

```powershell
.\gradlew.bat :app:compileDebugKotlin
```

## Примечания

Приложение использует документированное API YummyAnime и нативные компоненты Android для воспроизведения. Логика поиска видеопотоков реализована в этом репозитории и не копирует код других клиентов YummyAnime.

Релизные APK подписываются во время Gradle release-сборки и публикуются через GitHub Releases этого репозитория.

## Лицензия и контент

YummyDroid - клиентское приложение. Метаданные аниме и доступность видео зависят от YummyAnime и сторонних источников воспроизведения. Приложение не содержит и не распространяет аниме-контент.
