YummyDroid - неофициальный нативный клиент YummyAnime для Android, Android TV, телефонов, планшетов и ТВ-приставок.

## Возможности

- Каталог, топ, расписание, коллекции, библиотека и раздел загрузок.
- Поиск, голосовой поиск, сортировка и фильтры с мультивыбором.
- Авторизация, профиль, метки, избранное, оценки, комментарии и подписки на серии.
- Подробные карточки аниме: описание, постеры, скриншоты, жанры, студии, режиссёры, год, порядок просмотра, рекомендации и трейлеры.
- Нативный плеер Media3/ExoPlayer: выбор озвучки и качества, автоподбор рабочего источника, продолжение просмотра, PiP, пропуск опенинга/эндинга и автопереход к следующей серии.
- Проверка доступности доменов сайта и ручное управление списком доменов.
- Офлайн-просмотр: скачивание одной серии или всех серий аниме, выбор качества перед загрузкой, очередь загрузок и системное уведомление со статусом.
- Обновления через GitHub Releases.

## Офлайн-просмотр

Скачанные серии привязаны к конкретному аниме и конкретной серии. Если серия уже сохранена, приложение запускает локальный файл вместо онлайн-потока.

Вместе с видео сохраняются данные карточки аниме и список серий. Благодаря этому скачанное аниме можно открыть даже без доступа к сайту.

Если сайт или интернет недоступны, а на устройстве есть сохранённые серии, каталог автоматически переключается на офлайн-режим и показывает индикатор `Оффлайн`. При ручном включении фильтра `Доступно офлайн` такой индикатор не появляется.

APK доступен на странице релизов:

## Сборка

Требования:

- Android SDK.
- JDK 21.

Собрать release APK:

```powershell
.\gradlew.bat :app:assembleRelease
```

Проверить компиляцию Kotlin:

```powershell
.\gradlew.bat :app:compileDebugKotlin
```
