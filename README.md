YummyDroid — неофициальный нативный клиент YummyAnime для Android, Android TV, телефонов, планшетов и ТВ-приставок.

## Возможности

- Каталог, расписание, библиотека пользователя и раздел загрузок.
- Поиск, голосовой поиск, сортировка и фильтры с мультивыбором.
- Расписание с отдельным переключателем для скрытия прошедших релизов.
- Авторизация, профиль, метки, избранное, оценки, комментарии и подписки на серии.
- Подробные карточки аниме: описание, постеры, скриншоты, жанры, студии, режиссёры, год, порядок просмотра, рекомендации и трейлеры.
- Нативный плеер на Media3/ExoPlayer: выбор озвучки и качества, автоподбор лучшего источника, продолжение просмотра, PiP, пропуск опенинга/эндинга и автопереход к следующей серии.
- Масштабирование видео жестом в самом плеере.
- Проверка доступности доменов YummyAnime, автоматическое переключение и ручное управление списком доменов.
- Оффлайн-просмотр: скачивание одной серии или всех серий аниме, выбор озвучки и качества перед загрузкой, очередь загрузок и системное уведомление со статусом.
- Локальное сохранение карточки аниме и списка серий, если скачана хотя бы одна серия.
- Автоматический оффлайн-режим, когда интернет или сайт недоступны.
- Проверка обновлений через GitHub Releases.

## Оффлайн-просмотр

Скачанные серии привязаны к конкретному аниме и конкретной серии. Если серия уже сохранена, приложение запускает локальный файл вместо онлайн-потока.

Вместе с видео сохраняются данные карточки и список серий. Поэтому скачанное аниме можно открыть и смотреть даже без доступа к сайту.

Если сайт или интернет недоступны, а на устройстве есть сохранённые серии, каталог автоматически переключается на оффлайн-режим и показывает индикатор `Оффлайн`. При ручном включении фильтра `Доступно оффлайн` индикатор не показывается.

## Сборка

Требования:

- Android SDK.
- JDK 21.

Собрать release APK:

```powershell
.\gradlew.bat :app:assembleRelease
```

Проверить компиляцию Kotlin:

```powershell
.\gradlew.bat :app:compileDebugKotlin
```
